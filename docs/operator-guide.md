# Operator Guide

## First Deployment
1. Register operator, buildings/sites, chemical-handling and access
   scope, crews and robots.
2. Import existing client service and compliance history.
3. Run read-only access-scope and floor-scrubbing/vacuuming robot
   mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run follow-up record and audit export.

## Minimum Production Controls
- chemical-handling and access-scope validation before any dispatch
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (an out-of-scope
  chemical-handling task, an out-of-scope access request, an
  unverified follow-up record)
- evidence-backed follow-up records
- audit export for every dispatch, sign-off and follow-up record
- backup manual building-cleaning process

## Certification
Certified operators must prove robot-safety integrity, chemical-
handling/access-scope discipline, evidence-backed follow-up records
and human review for dispatch-affecting actions.
